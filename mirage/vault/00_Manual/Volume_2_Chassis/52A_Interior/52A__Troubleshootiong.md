TROUBLESHOOTING 52300280036

DIAGNOSTIC FUNCTION

INPUT SIGNAL INSPECTION POINTS <VEHICLES WITH ETACS-ECU>

When Using the Scan Tool I

1.

Connect the scan tool to the data link connector.

Caution The scan tool should be connected or disconnected after turning the ignition switch to the OFF position.

2.

If buzzer of the scan tool sounds once when the front seat belt buckle switch is operated (ON/OFF), the ETACS- ECU input signal for that switch circuit system is normal.

When Using the Voltmeter

Ground terminal

1.

Use the special tool to connect a voltmeter between the ground terminal and the ETACS terminal of the data link connector.

2.

If the voltmeter indicator deflects once when the front seat belt buckle switch is operated (ON/OFF), the ETACS- ECU input signal for that switch circuit system is normal.
