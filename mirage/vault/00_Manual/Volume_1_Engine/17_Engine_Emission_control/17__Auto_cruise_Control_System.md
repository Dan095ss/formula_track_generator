METHOD OF ERASING DIAGNOSTIC TROUBLE CODES

1 procedure.

rase the diagnostic trouble codes with the following

NOTE The diagnostic trouble codes will not be erased even if the battery (-) terminal is disconnected.

/

Using the scan tool Caution To prevent damage to the scan tool, make sure the ignition switch is “OFF” before connecting or disconnecting the scan tool.

1. Turn the ignition switch “OFF.”

2. Connect scan tool to the data link connector.

3. Use scan tool to check for auto-cruise control system diagnostic trouble codes.

4. Turn the ignition switch “OFF.”

5. Disconnect the scan tool.

INPUT SWITCH CODE CHECK METHOD

1. Connect the scan tool to the data link connector (16-pin) under the instrument panel under cover.

2. Turn the ignition switch to ON.

3. After pushing the auto-cruise control switch in the direction of arrow (6) in the illustration, press the cruise control main switch to the ON position, and within 1 second after doing this, push the cruise control switch back in the direction of arrow (A).

4. Operate each switch listed in the input check table and take a reading of the input switch codes with the scan tool.
