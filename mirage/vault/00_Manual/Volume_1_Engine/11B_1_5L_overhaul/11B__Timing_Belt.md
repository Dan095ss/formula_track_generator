REMOVAL SERVICE POINTS +AF TIMING BELT/TENSIONER SPRING/TIMING BELT TENSIONER REMOVAL

(1) Using pliers, grip the tensioner spring end (marked “A’ in the illustration) and remove it from the oil pump case lug. Then, remove the tensioner spring.

(2) Remove the timing belt tensioner, and then remove the timing belt.

(3) If the timing belt is to be reused, chalk an arrow on the belt to indicate the direction of rotation before removing it. This will ensure the timing belt is fitted correctly when reused.

+BF CAMSHAFT SPROCKET BOLT REMOVAL

(1) Using the special tools shown in the illustration, lock the camshaft sprocket in position.

(2) Loosen the camshaft sprocket bolt.

INSTALLATION SERVICE POINTS F A + CAMSHAFT SPROCKET BOLT INSTALLATION

(1) Using the special tools shown in the illustration, lock the camshaft sprocket in the position.

(2) Tighten the camshaft sprocket bolt to the specified torque.
