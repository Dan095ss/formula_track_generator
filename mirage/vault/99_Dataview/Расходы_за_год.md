# 💰 Расходы за последние 12 месяцев

## Сводка по типам работ

```dataview
TABLE WITHOUT ID
  service_type AS "Тип работы",
  length(rows) AS "Случаев",
  sum(rows.cost_total_rub) AS "Сумма, руб"
FROM "10_Maintenance"
WHERE type = "maintenance"
  AND status = "done"
  AND date >= date(today) - dur(365 days)
GROUP BY service_type
SORT sum(rows.cost_total_rub) DESC
```

## Все записи за год

```dataview
TABLE WITHOUT ID
  file.link AS "Работа",
  date AS "Дата",
  mileage_km AS "Пробег",
  service_type AS "Тип",
  cost_parts_rub AS "Запчасти",
  cost_labor_rub AS "Работы",
  cost_total_rub AS "Итого"
FROM "10_Maintenance"
WHERE type = "maintenance"
  AND status = "done"
  AND date >= date(today) - dur(365 days)
SORT date DESC
```
