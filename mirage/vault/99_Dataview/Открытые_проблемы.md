# ⚠️ Открытые и активные проблемы

## По серьёзности

```dataview
TABLE WITHOUT ID
  file.link AS "Проблема",
  date_reported AS "Обнаружено",
  mileage_km AS "Пробег",
  severity AS "Серьёзность",
  status AS "Статус",
  suspected_systems AS "Системы"
FROM "20_Issues"
WHERE type = "issue"
  AND (status = "open" OR status = "diagnosing" OR status = "waiting_parts")
SORT
  choice(severity = "critical", 0, choice(severity = "high", 1, choice(severity = "medium", 2, 3))) ASC,
  date_reported ASC
```

## Сводка по статусам

```dataview
TABLE WITHOUT ID
  status AS "Статус",
  length(rows) AS "Сколько"
FROM "20_Issues"
WHERE type = "issue"
GROUP BY status
SORT status ASC
```

## Решённые за последние 90 дней

```dataview
TABLE WITHOUT ID
  file.link AS "Проблема",
  resolved_date AS "Решена",
  cost_to_resolve_rub AS "Стоило, руб",
  resolution_maintenance AS "Ремонт"
FROM "20_Issues"
WHERE status = "resolved"
  AND resolved_date >= date(today) - dur(90 days)
SORT resolved_date DESC
```
