# 🛢 История замен масла

```dataview
TABLE WITHOUT ID
  file.link AS "Запись",
  date AS "Дата",
  mileage_km AS "Пробег, км",
  cost_total_rub AS "Стоило, руб",
  shop AS "Где",
  notes_short AS "Заметки"
FROM "10_Maintenance"
WHERE type = "maintenance"
  AND service_type = "oil_change"
  AND status = "done"
SORT date ASC
```

## Интервал между заменами

```dataviewjs
const pages = dv.pages('"10_Maintenance"')
  .where(p => p.type === "maintenance"
            && p.service_type === "oil_change"
            && p.status === "done"
            && p.mileage_km)
  .sort(p => p.date, "asc")
  .array();

if (pages.length < 2) {
  dv.paragraph("Нужно как минимум 2 замены, чтобы посчитать интервалы.");
} else {
  const rows = [];
  for (let i = 1; i < pages.length; i++) {
    const km = (pages[i].mileage_km ?? 0) - (pages[i - 1].mileage_km ?? 0);
    const days = pages[i].date && pages[i - 1].date
      ? Math.round((pages[i].date.toMillis() - pages[i - 1].date.toMillis()) / 86_400_000)
      : "?";
    rows.push([
      pages[i].file.link,
      pages[i - 1].mileage_km, pages[i].mileage_km,
      km, days,
    ]);
  }
  dv.table(
    ["Запись", "С пробега", "До пробега", "Интервал, км", "Интервал, дней"],
    rows
  );
}
```

## Следующая замена
